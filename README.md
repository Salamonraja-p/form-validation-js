# Form Validation using JavaScript

This project is a simple form validation application built using HTML, CSS, and JavaScript.  
It validates user input such as username, email, password, and confirm password before form submission.

## Features
- Input validation using JavaScript
- Error messages for invalid fields
- Password and confirm password matching
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event handling
- Form validation logic
- Basic use of regular expressions
